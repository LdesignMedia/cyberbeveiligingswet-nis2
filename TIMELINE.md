# NIS2 / Cbw Timeline

Key dates for the NIS2 Directive, the Commission Implementing Regulation (CIR 2024/2690), the CER Directive, the Dutch Cyberbeveiligingswet (Cbw), and the Wet weerbaarheid kritieke entiteiten (Wwke). Sister regulations covering AMvB / besluit are folded in.

Dates marked **(provisional)** are based on parliamentary planning and may shift. All others are confirmed by the cited source.

---

## EU layer

### Pre-NIS2 background

| Date | Event |
|------|-------|
| **2016-07-06** | NIS1 Directive (EU) 2016/1148 adopted — first EU-wide cybersecurity baseline. |
| **2018-05-10** | NIS1 transposition deadline. NL transposed via Wbni. |
| **2020-12-16** | Commission proposal for NIS2 published, alongside CER Directive proposal. |

### NIS2 Directive (EU) 2022/2555

| Date | Event |
|------|-------|
| **2022-12-14** | NIS2 adopted by European Parliament + Council. |
| **2022-12-27** | Published in Official Journal L 333. |
| **2023-01-16** | NIS2 enters into force (20 days after publication). |
| **2024-10-17** | NIS2 transposition deadline for Member States. |
| **2024-10-18** | NIS1 (2016/1148) repealed. |
| **2025-04-17** | Member States must produce list of essential + important entities (NIS2 art. 3(3)). |

### CIR — Commission Implementing Regulation (EU) 2024/2690

| Date | Event |
|------|-------|
| **2024-10-17** | CIR 2024/2690 adopted. |
| **2024-11-07** | Published in Official Journal. |
| **2024-11-27** | Enters into force (20 days after publication). |
| **2024-10-17** | Direct-effect requirements bind digital infrastructure entities (DNS, TLD, cloud, datacenter, CDN, MSP, MSSP, online marketplaces, search engines, social networks, trust services). |

### CER Directive (EU) 2022/2557 — sister regime

| Date | Event |
|------|-------|
| **2022-12-14** | CER adopted alongside NIS2. |
| **2023-01-16** | Enters into force. |
| **2024-10-17** | CER transposition deadline. |
| **2026-07-17** | Member States must designate critical entities (CER art. 6). |

### ENISA + Cooperation Group

| Date | Event |
|------|-------|
| **2025-05** | ENISA publishes *Handbook for Cyber Stress Tests v2.0*. |
| **2025-06** | ENISA publishes *Technical Implementation Guidance v1.0* mapping CIR 2024/2690 to practical measures. |
| **2025-06** | ENISA publishes *Cybersecurity Roles and Skills for NIS2 Essential and Important Entities* (ECSF mapping). |

## NL layer — Cyberbeveiligingswet (Cbw)

### Drafting + consultation

| Date | Event |
|------|-------|
| **2023** | Pre-consultation drafts circulated by Ministerie van J&V. |
| **2024** | Wetsvoorstel 36764 (Cbw) submitted to Tweede Kamer. Note: NL missed the EU 2024-10-17 deadline. |
| **2025-02-20** | Internetconsultatie *Cyberbeveiligingsbesluit* (AMvB onder Cbw) opens. |
| **2025-03-30** | Internetconsultatie Cyberbeveiligingsbesluit closes. |

### Adoption

| Date | Event |
|------|-------|
| **2025** | Memorie van toelichting (kamerstuk 2025D26060) + nota van wijziging (2025D45516) processed. |
| **2026-04-15** | **Tweede Kamer adopts Cbw (36764) and Wwke (36765).** |
| **2026-Q2** | Eerste Kamer adoption expected (provisional). |
| **~2026-07-01** | Cbw + Wwke entry into force expected (provisional). |

### Post-entry

| Date | Event |
|------|-------|
| **+0 days from entry into force** | Wbni repealed; Cbw zorgplicht + meldplicht + registratieplicht active. |
| **+0 days from entry into force** | Cyberbeveiligingsbesluit (AMvB) and ministeriële regeling expected to enter into force same day. |
| **+90–180 days (provisional)** | Initial registration window for in-scope entities via NCSC-NL portal. |
| **+12 months (provisional)** | Board-training duty (art. 20 NIS2 → Cbw zorgplicht art.) operational milestone. |

## NL layer — Wwke (sister track)

| Date | Event |
|------|-------|
| **2024** | Wetsvoorstel 36765 (Wwke) submitted to Tweede Kamer. |
| **2025** | Internetconsultatie *Besluit Wwke* runs. |
| **2026-04-15** | Tweede Kamer adopts Wwke. |
| **~2026-07-01** | Wwke entry into force expected (provisional). |
| **By 2026-07-17** | NL must designate critical entities under CER art. 6. |
